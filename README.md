# 三味線譜変換ツール

五線譜（PDF）を三味線の勘所情報付き中間YAMLに変換するツール。
Google Colab 上で全工程を実行できる。

---

## 処理の流れ

```
PDF → MusicXML（Audiveris）→ 三味線中間YAML（shamisen_converter）
```

9. **セル3-a〜c** で三味線中間YAMLに変換
10. **セル4** でダウンロード

## 設計上の決定事項

### 対象
- 流派：文化譜（長唄系・最もメジャー）

### 調弦
- 本調子 / 二上り / 三下り の3種類に対応（六本相当の絶対音高で定義）
- 実行時にユーザーが選択する

### 弦・勘所の優先順位
- 同じ音が複数の弦で出せる場合、**開放弦からの半音数が最小のポジションを優先**
- 同点の場合は 三の糸 → 二の糸 → 一の糸 の順

### 音域外の音
- 警告を出してユーザーに処理を選ばせる
- 選択肢：オクターブ上げ／下げ／スキップ／未解決のまま

### 和音
- 三味線は単音楽器のため、和音は最高音のみ使用

## 勘所マッピング（shamisen_mapping.yaml）

参考：https://www.asahi-net.or.jp/~hb9t-ktd/music/Japan/Instrument/shamisen_score.html

勘所番号は**半音間隔**。数字の間に記号が挿入される：
- `#` … 3番と4番の間（+4半音）
- `b` … 9番と10番の間（+11半音）
- `1#` … 13番と14番の間（+16半音）

各弦ともに 0〜18 + 記号3個 = **22ポジション（0〜21半音）**

### 本調子（三=D4, 二=G4, 一=D5）

| 勘所 | 三の糸 | 二の糸 | 一の糸 |
|:----:|--------|--------|--------|
| 0    | D4     | G4     | D5     |
| 1    | D#4    | G#4    | D#5    |
| 2    | E4     | A4     | E5     |
| 3    | F4     | A#4    | F5     |
| #    | F#4    | B4     | F#5    |
| 4    | G4     | C5     | G5     |
| 5    | G#4    | C#5    | G#5    |
| 6    | A4     | D5     | A5     |
| 7    | A#4    | D#5    | A#5    |
| 8    | B4     | E5     | B5     |
| 9    | C5     | F5     | C6     |
| b    | C#5    | F#5    | C#6    |
| 10   | D5     | G5     | D6     |
| 11   | D#5    | G#5    | D#6    |
| 12   | E5     | A5     | E6     |
| 13   | F5     | A#5    | F6     |
| 1#   | F#5    | B5     | F#6    |
| 14   | G5     | C6     | G6     |
| 15   | G#5    | C#6    | G#6    |
